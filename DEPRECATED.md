# API Deprecate

Questa pagina raccoglie API e servizi che erano precedentemente inclusi nella lista principale ma che sono stati rimossi perché:

- Non più mantenuti o ufficialmente ritirati

Mantenere questa lista serve a dare contesto storico e a evitare che risorse già valutate vengano ripresentate in PR future.
